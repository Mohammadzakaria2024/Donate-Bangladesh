What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for the web documents.In an HTML document,DOM allows JavaScript to access and modify the contents of the page.This is how interactive features on websites are implemented.

There are many ways to select an element:

  • 1.getElementById()
  • 2.getElementsByClassName()
  • 3.getElementsByTagName()
  • 4.querySelector()
  • 5.querySelectorAll()

    What is event delegation in the context of the DOM, and why is it useful? Event delegation is a technique in JS that set event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.Instead of adding individual event listeners to multiple child elements,one can add a single event listener to a parent element.This parent element can then manage all events that bubble up from its child elements.
